# Flux Alfred Control (plus remote)

Control Flux using Alfred, or Alfred Remote.

### Features:

* Trigger preferences
* Quit Flux
* Trigger Movie Mode
* Trigger Darkroom Mode
* Disable Flux for 1 hour
* Disable Flux till sunrise
* Disable Flux for current app
